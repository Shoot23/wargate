WARGATE is a browser-based RTS and autobattler, built solo as a side project: ~31,000 lines of vanilla HTML and JavaScript, no engine, no framework. It has two full campaign missions and a skirmish mode, with custom systems for unit AI and targeting, battle flow, economy, and mission scripting.

I built it the way I now build everything, with AI as a genuine second brain rather than an autocomplete: Claude Code working against the full codebase, structured project context, and my own review and design judgment on everything that ships. The project is partly a game and partly a proving ground for that workflow. It's where I've tested how far AI-assisted development scales as a codebase grows, what breaks when context gets large, and what a non-engineer with a computer science background can realistically ship alone. The short answer: a lot more than the job title "product manager" used to imply.

Play it in the browser: no install, no account.
